## Overview
This repository contains a collection of machine learning and data analysis projects. These tasks demonstrate practical applications of classification, time-series forecasting, clustering, and collaborative filtering.

---

## 📂 Project Directory

### Task 1: Titanic Survival Prediction (Binary Classification)
* **Objective:** Predict passenger survival rates using historical data.
* **Techniques Used:** Data Imputation, Feature Engineering (One-Hot Encoding), Random Forest Classification.
* **Performance:** Evaluated using precision, recall, and accuracy scores.

### Task 2: Stock Price Forecasting (Time-Series Analysis)
* **Objective:** Forecast future stock closing prices based on historical trading volumes and price points.
* **Techniques Used:** MinMax Scaling, Sequence Generation, LSTM (Long Short-Term Memory) Neural Networks.

### Task 3: Customer Segmentation (Unsupervised Learning)
* **Objective:** Identify distinct consumer profiles based on purchasing behavior and annual income.
* **Techniques Used:** The Elbow Method, K-Means Clustering, 2D Scatter Visualization.

### Task 4: Movie Recommendation System (Collaborative Filtering)
* **Objective:** Recommend movies to users based on correlation with their previously highly-rated films.
* **Techniques Used:** Sparse Matrix generation, Pearson Correlation, Data Merging.
